## Check

Check is a simple terminal based to-do app implemented in Python

## Usage

Simply run:

```
python3 main.py
```

Check will automatically create the needed files. There are no external libraries. Your tasks automatically get saved.

### Commands

| Key        | Command     |
|:----------:|:-----------:|
| q          | exit        |
| j / <down> | down        |
| k / <up>   | up          |
| <space>    | new task    |
| <enter>    | (un)check   |
| d          | delete task |
| e          | edit task   |
